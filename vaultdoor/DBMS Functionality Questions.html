Question 27: "What is the main difference between an INNER JOIN and an OUTER JOIN in SQL?"

  • The correct answer is a. INNER JOIN returns only the records where the joined keys are in both specified tables, OUTER JOIN returns all records from one table and any records in other table match the joined keys. An INNER JOIN only includes rows where there is a match in both tables, whereas an OUTER JOIN includes all rows from one or both tables, even if there is no match.
